Graph
=====
Задаётся большой интервал (A, B) c шагом прохода (H) и находим отрезки, где есть корень. 
Далее, уточняем с точностью по X (EPSILON). Есть ограничение на итерации(ItM)
Шаг такой, что на нём всегда максимум 1 корень.
Строится таблица (stringrid):
  
| Name                             | Description                             |
|----------------------------------|-----------------------------------------|
| Номер корня                      | 1..8                                    |
| Концы отезка c корнем            | (X-H, X)/ (X, X+H)                      |
| Корень                           | 6-9 цифр, спецификация e                |
| Значение функции в корне         | Спецификация е, минимум цифр в мантиссе |
| Количество совершенных итераций  | Если максимум и больше, то  ItM. (-)    |
| Код ошибки. Вывод перечня ошибок | Кнопка Help?/Описание при наведении?    |

Выбор из 3 фунций(Combobox). 2 простые. 1 Своя. 
Нарисовать график с  выделением корня.
